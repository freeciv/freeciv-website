Download Freeciv

Here you can download the latest version of Freeciv. To learn what's changed between releases, see the NEWS page. For more download distributions, see the wiki.

Stable 2.6

See the list of changes in the most recent stable release, 2.6.6.

Source Code

To compile Freeciv yourself, download the source code:
  • freeciv-2.6.6.tar.bz2
    Mirror 1 (sourceforge.net)
    Mirror 2 (freeciv.org)
  • freeciv-2.6.6.zip
    Mirror 1 (sourceforge.net)
    Mirror 2 (freeciv.org)
There are listings of checksums for these source code archives, current and past, at http://files.freeciv.org/stable/SHA256SUM and SHA1SUM and MD5SUM.

Windows Packages

Windows binaries are available at:
  • Freeciv-2.6.6-win32-gtk3-setup.exe Mainstream client, with map editor
    Mirror 1 (sourceforge.net)
    Mirror 2 (freeciv.org)
  • Freeciv-2.6.6-win32-qt-setup.exe Alternative mainstream client
    Mirror 1 (sourceforge.net)
    Mirror 2 (freeciv.org)
  • Freeciv-2.6.6-win32-sdl-setup.exe Experimental and more immersive client
    Mirror 1 (sourceforge.net)
    Mirror 2 (freeciv.org)
Pick one. You can play the same Freeciv games with them; only the user interface is different.
