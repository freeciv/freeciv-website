Download Freeciv

Here you can download the latest version of Freeciv. To learn what's changed between releases, see the NEWS page. For more download distributions, see the wiki.

Stable 3.1

See the list of changes in the most recent stable release, 3.1.3.

Source Code

To compile Freeciv yourself, download the source code:
  • freeciv-3.1.3.tar.xz
    Mirror 1 (sourceforge.net)
    Mirror 2 (freeciv.org)
  • freeciv-3.1.3.zip
    Mirror 1 (sourceforge.net)
    Mirror 2 (freeciv.org)
There are listings of checksums for these source code archives, current and past, at https://files.freeciv.org/stable/SHA256SUM and SHA1SUM and MD5SUM.

Windows Packages

Windows binaries for Windows 10 or later are available at:
  • Freeciv-3.1.3-msys2-win64-10-gtk3.22-setup.exe Mainstream client, with map editor
    Mirror 1 (sourceforge.net)
    Mirror 2 (freeciv.org)
  • Freeciv-3.1.3-msys2-win64-10-qt6-setup.exe Alternative mainstream client
    Mirror 1 (sourceforge.net)
    Mirror 2 (freeciv.org)
  • Freeciv-3.1.3-msys2-win64-10-gtk4-setup.exe Future Mainstream client, with map editor
    Mirror 1 (sourceforge.net)
    Mirror 2 (freeciv.org)
  • Freeciv-3.1.3-msys2-win64-10-sdl2-setup.exe Experimental and more immersive client
    Mirror 1 (sourceforge.net)
    Mirror 2 (freeciv.org)
  • Freeciv-ruledit-3.1.3-msys2-win64-10-setup.exe Freeciv ruleset editor
    Mirror 1 (sourceforge.net)
    Mirror 2 (freeciv.org)
Pick one. You can play the same Freeciv games with them; only the user interface is different.
